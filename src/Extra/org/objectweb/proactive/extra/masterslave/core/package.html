Core classes of the Master/Slave API. The package includes :

  • AOMaster : the master Active Object
  • AOSlaveManager : the Active Object which handles Slave deployment
  • AOSlave : the active objects which will execute the tasks
  • AOPinger : the active object taking care that all slaves are alive
  • TaskWrapperImpl : an internal version of the user tasks
